# 🏥 Acil Servis Nöbet Sistemi

Web tabanlı nöbet planlama ve yönetim sistemi.

## Özellikler
- ✅ Sürükle-bırak nöbet atama
- ✅ Excel raporları (Nöbet Listesi, Puantaj)
- ✅ Fazla mesai hesaplama
- ✅ Tatil günü işaretleme
- ✅ Kullanıcı yetkilendirmesi (Admin/Pratisyen)

## Teknolojiler
- Backend: Flask (Python)
- Frontend: React (Babel)
- Veritabanı: In-memory (geliştirme aşamasında)

## Kurulum
```bash
pip install -r requirements.txt
python app.py
```

## Kullanıcılar
- Yönetici: Admin yetkisi ile tam kontrol
- Pratisyen: Sadece görüntüleme yetkisi
